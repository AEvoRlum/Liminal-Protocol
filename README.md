# Future Update Plans

+ **A complete, high-quality campaign independent of the vanilla campaign, including tech tree, turrets, unit designs, etc.**

+ **Contingency Contract: if you have played the game Arknights, you will know this is a challenge mode more terrifying than a nightmare. Players select modifiers (enemy enhancements and player debuffs) to determine difficulty and score. Upon clearing a stage, players will earn a "Medal" (also from Arknights). Different medal designs are awarded based on the score, proving the player's mental prowess or strong perseverance. There is no need to be upset by failure; this is merely a challenge mode (it does not affect performance in the campaign).**

+ **Unit Brawling, or combat betting: most Chinese children love this type of gameplay. Players face two units with different "combat power" and predict which side will win, voting for the side they believe will be victorious. Correct guesses increase the player's ranking within the round; too many incorrect guesses result in elimination (obviously a multiplayer mode, currently only being considered for development).**
